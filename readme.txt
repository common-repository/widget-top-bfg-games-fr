== Description ==

Ce plugin est simple: un widget qui affiche une liste des meilleurs jeux à partir de votre barre latérale BigFish widgetized sur la langue française seulement (.fr).
C'est très utile si vous voulez créer un site pour des langues différentes et voulez utiliser tous les sommets de jeux pour les différentes langues.

ENGLISH:
This simple plugin is a widget that displays a list of top casual games from BigFish on your widgetized sidebar on French language only (.fr).
It's very useable if you want to create one site for different languages and want to use all tops of games for different languages.

== Installation ==

1. Le dossier widget de téléchargement contient des 'widget-top-BFG-jeux-FR.php' au '/wp-content/plugins/' répertoire.
2. Activez le plugin dans le menu 'Plugins' dans WordPress.
3. Utilisez votre 'Présentation'/'Widgets Sidebar' section pour placer le widget et de le configurer.

ENGLISH:
1. Upload widget folder contains 'widget-top-bfg-games-FR.php' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use your 'Presentation'/'Sidebar Widgets' section to place the widget and to configure it.

== Configuration ==

* Widget titre: le titre du widget
* Nombre de Jeux d'affichage: Nombre de jeux dessus vous souhaitez être affiché (1-10)
* Type de jeux à afficher: Types de jeux que vous souhaitez afficher (jeux PC / Jeux en ligne)
* Texte et images Aligner: Alignez la liste des jeux que vous souhaitez afficher (par défaut le thème / gauche / centre / droite)
* Voir la liste des balles: Est-ce besoin de montrer des balles liste (utilisez un style à thème)
* Games Show Position: Est-ce besoin de montrer la position des jeux dans la liste en haut
* Afficher les images des Jeux: Est-ce besoin de montrer des images des jeux dans plus de titre de jeu
* Type d'images: images de type que vous souhaitez afficher (petites 60x40 / 80x80 par défaut / vedette 175x150 / 175x150 subfeatured)
* Games Show Text: est nécessaire pour afficher le texte des jeux dans plus de titre de jeu et de l'image
* Qu'est-dessus: Qu'est-dessus vous souhaitez être affiché - texte ou image
* Votre identifiant BigFishGames: Vous pouvez laisser ce champ vide, un widget peut travailler sans connexion, ou utiliser la connexion correcte.
* Votre canal client BigFishGames: nom de domaine sera utilisé par défaut. Laissez ce champ vide, de ne pas utiliser les canaux. Alpha numérique uniquement.

ENGLISH:
* Widget title: the title of the widget
* Number of Games to Display: Number of top games you would like to be displayed (1-10)
* Games Type to Display: Games Types you would like to be displayed (PC games / Online Games)
* Text and Images Align: Align of games list you would like to be displayed (theme default / left / center / right)
* Show List Bullets: Is need to show list bullets (use theme style)
* Show Games Position: Is need to show games position in top list 
* Show Games Images: Is need to show games images in addition to game title
* Images Type: Images Type you would like to be displayed (small 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Games Text: Is need to show games text in addition to game title and image
* What Above: What Above you would like to be displayed - Text or Image
* Your BigFishGames login: You can leave this field empty, widget can work without login, or use correct login.
* Your BigFishGames client channel: Domain name will be used by default. Leave this field empty, to not use channels. Alpha numeric only.
